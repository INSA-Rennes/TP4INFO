What's new:

  • Eclipse applications written with Kepler can now be tested.
  • HTML test result reports offer on-click expansion of screenshots taken on errors
  • Search scopes are supported in the search dialog.
  • There are new actions available to prepare for AUT termination and synchronize the re-starting of an AUT that is closed during the test.
  • There are new functions available for accessing some node properties, and for accessing specific cells from within central test data sets.

The list above is just an overview. For a more complete guide to new features, read the release notes.
